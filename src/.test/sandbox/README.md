# Sandbox Archives

This directory stores source snapshots used for sandbox-style comparisons and historical test fixtures.

- `base/` keeps the original legacy sandbox snapshot.
- `2026-06-06/` keeps the repository snapshot taken on 2026-06-06.

For future sandbox archives, create a new top-level folder using the date format `YYYY-MM-DD`.
Keep `base/` reserved for the original baseline snapshot.
